# Extended Watchlist & Honorable Mentions

> This document contains projects that are highly relevant to the AI and Agent ecosystem but did not make the strict 6-8 item cut for the main `README.md` list. This includes foundational models, specialized toolkits, and rising experiments from late March 2026.
> 
> Last reviewed: March 27, 2026

## 1. Multi-Agent & Enterprise Frameworks

| Project | Why it matters |
|---|---|
| **[Microsoft Agent Framework (AutoGen 0.4)](https://github.com/microsoft/autogen)** | The unified, enterprise-grade evolution of AutoGen and Semantic Kernel, featuring asynchronous event-driven architecture. Highly relevant but currently undergoing heavy migration from previous AutoGen versions. |
| **[MetaGPT](https://github.com/geekan/MetaGPT)** | Simulates a full software company (PM, engineers, QA) using multi-agent role-playing. Still popular with over 61,000 stars. |
| **[Deer-Flow 2.0](https://github.com/bytedance/deer-flow)** | ByteDance's newest agent orchestration framework that rocketed past 35,000 stars in March 2026, leveraging LangGraph for modular collaborations. |
| **[Maestro](https://github.com/maestro-ai/maestro)** | An Agent Orchestration Command Center allowing fleets of AI agents to be run in parallel, using Git worktrees for isolated task execution. |

## 2. Specialized Agent Toolkits & Memory

| Project | Why it matters |
|---|---|
| **[Haystack](https://github.com/deepset-ai/haystack)** | Specializes in building production-ready RAG pipelines and AI-powered search, excelling in automated research and document retrieval. |
| **[Claude-Mem](https://github.com/claude-mem/claude-mem)** | Provides persistent memory architecture specifically tailored for Claude Code, utilizing a secure hook-based system. |
| **[Agent-Lightning](https://github.com/microsoft/agent-lightning)** | A Microsoft Agent Trainer framework optimized for improving agent behavior through reinforcement learning and prompt optimization. |

## 3. Terminal & Development Utilities

| Project | Why it matters |
|---|---|
| **[GitHub Spec Kit](https://github.com/github/spec-kit)** | Innovating in "Spec-Driven Development", using AI to generate and maintain complex application code entirely from structured markdown specifications. |
| **[VibeTunnel](https://github.com/vibetunnel/vibetunnel)** | Transforms any standard web browser into a terminal for remote agent monitoring, featuring session recording and Git follow modes. |
| **[Pretext](https://github.com/chenglou/pretext)** | A pure JS/TS library for multiline text measurement without the DOM. Highly useful for AI agents performing UI measurement, layout calculation, and text iteration without triggering expensive browser reflows. |

| Project | Why it matters |

## How projects move between lists

The AI landscape changes quickly. Projects in this `EXTENDED_WATCHLIST.md` may be promoted to the main `README.md` if they show sustained, explosive growth or become undeniable defaults in their respective categories. Conversely, projects in the main repository that lose momentum or are superseded by newer paradigms may be archived here.
